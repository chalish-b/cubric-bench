## runs-v1

All runs collected against the first PLL suite (`suites/_archived/pll-v1`),
2026-06-11. **Not comparable** to current or future runs: they were scored
against the old algorithms and the random-AUF variant scheme (see
`suites/_archived/README.md`).

These were early harness-shakedown runs on mostly free/cheap models
(gpt-oss-20b/120b, owl-alpha, deepseek-v4-flash, gemini-3/3.5-flash, gpt-5.4).
Kept for reference and as test data for building the results aggregation, but
they must not be mixed into any leaderboard.

Each run's `run.json` still carries its original `suiteHash`, which no longer
matches any live suite — so the harness already flags them as stale.
